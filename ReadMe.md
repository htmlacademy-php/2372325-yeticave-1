**Обратите внимание на файл:**
- `Contributing.md` — руководство по внесению изменений;
- Не удаляйте файлы: `.editorconfig`, `.gitattributes`, `.gitignore`.

### Установка приложения
- на основе файла `config.sample.php` создать файл `config.php`;
- вписать в `config.php` свои настройки конфигурации;
- накатить структуру БД, выполнив sql запросы из файла `sql/schema.sql`.
